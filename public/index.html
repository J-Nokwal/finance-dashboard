Core capabilities

Everything you need to power a multi-tenant financial platform

🔐
Multi-strategy Auth
Email/password, one-time-password (OTP), and Google OAuth — all under one unified session management system with JWT refresh tokens.
🏢
Multi-tenant Organizations
Full organization lifecycle with soft-delete support. Each org can hold multiple projects and members with isolated access control.
🛡️
Role-Based Access Control
Granular RBAC at both org (Owner / Admin / Member) and project (Owner / Admin / Analyst / Viewer) levels with middleware enforcement.
📬
Invitation System
Token-based invitations with expiry, role assignment, and project-level scoping — all delivered via transactional email through Resend.
📊
Project Management
Nested projects within organizations with per-project membership. Soft-delete, audit trails, and member role management built in.
📖
Auto-generated Swagger Docs
Full OpenAPI specification served at /docs — explore every endpoint interactively without any extra tooling.
